# Support Page

This is a simple support page template designed for App Store submissions and general app support.

## Usage

1. Open `index.html`.
2. Replace `[Your App Name]`, `[Your Company Name]`, and `support@yourdomain.com` with your actual details.
3. Replace the placeholder icon div with an `<img>` tag pointing to your app icon if desired.

## Deployment

This site is configured to deploy to GitHub Pages using GitHub Actions.

1. Push this repository to GitHub.
2. Go to **Settings** > **Pages** and set **Source** to **GitHub Actions**.
3. Automatic deploys run on every push to `main`.
4. Manual deploys are available from **Actions** > **Deploy GitHub Pages** > **Run workflow**.
5. Your site will be live at `https://username.github.io/repo-name/`.
